# Development Tests

This directory contains tests created during specific development phases:

- `test_clean_constants.py` - Clean constants implementation tests
- `test_more_constants.py` - Additional constants testing
- `test_create_method.py` - Creation method testing
- `test_unit_arithmetic.py` - Unit arithmetic functionality tests
- `test_unit_arithmetic_final.py` - Final unit arithmetic tests
- `test_unit_arithmetic_fixed.py` - Fixed unit arithmetic tests

## Purpose

These tests represent development iterations and may include:
- Work-in-progress functionality
- Multiple versions of similar tests
- Development milestone validation
- Feature development testing

## Running Development Tests

```bash
# Run specific development test
python tests/development/test_clean_constants.py
python tests/development/test_unit_arithmetic.py
```

**Note**: Some tests may be outdated or superseded by newer implementations.
