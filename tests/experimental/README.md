# Experimental Tests

This directory contains experimental, debug, and quick test files used during development:

- `debug_test.py` - Debug testing and troubleshooting
- `debug_unit_symbols.py` - Debug unit symbol functionality
- `quick_test.py` - Quick testing during development
- `quick_test2.py` - Additional quick testing

## Purpose

These tests are used for:
- Debugging specific issues
- Quick verification during development
- Experimental feature testing
- Troubleshooting problems

## Running Experimental Tests

```bash
# Run specific experimental test
python tests/experimental/debug_test.py
python tests/experimental/quick_test.py
```

**Note**: These tests may not be stable or may require specific conditions to run successfully.
